# Баг-репорты Excursium

## Баг №1: Фильтр «Место проведения» не сохраняется при переходе на главную страницу и обратно

| Поле | Значение |
|------|----------|
| **Страница** | `https://excursium.com/ekskursii-dlya-shkolnikov/list` |
| **Серьёзность** | Незначительная (Minor) |
| **Приоритет** | Средняя (Medium) |
| **Окружение** | Windows 10, Google Chrome 147 |

**Шаги для воспроизведения:**
1. Открыть страницу со списком экскурсий.
2. Выбрать в фильтре «Место проведения» значение «Москва».
3. Нажать на логотип «Экскурсиум» в шапке сайта (переход на главную).
4. Снова перейти в раздел «Все экскурсии» (`/ekskursii-dlya-shkolnikov/list`).

**Фактический результат:**  
Фильтр «Москва» сброшен, отображаются все экскурсии.

**Ожидаемый результат:**  
Выбранный фильтр сохраняется (например, в localStorage или URL), список экскурсий остаётся отфильтрованным по Москве.

## Баг №2: Поля «Телефон» и «Email» в форме обратной связи не имеют масок ввода и валидации

| Поле | Значение |
|------|----------|
| **Страница** | `https://excursium.com/About/Contact` |
| **Серьёзность** | Основная (Major) |
| **Приоритет** | Высокая (High) |
| **Окружение** | Windows 10, Google Chrome 147 |

**Шаги для воспроизведения:**
1. Открыть страницу «Контакты».
2. В поле «Телефон» ввести:
   - буквы (например, `abcde`)
   - или номер без кода страны (например, `9123456789`)
3. В поле «Email» ввести:
   - текст без `@` (например, `testmail.ru`)
   - или просто буквы (например, `abcdef`)
4. Отправить форму.

**Фактический результат:**  
Форма принимает любой ввод без проверки формата. Маска для телефона отсутствует. Email не проверяется на наличие `@` и домена. Сообщение отправляется с некорректными данными.

**Ожидаемый результат:**  
- Поле «Телефон» должно иметь маску ввода для российского номера (`+7 (XXX) XXX-XX-XX`).
- Поле «Email» должно проверять формат (наличие `@` и домена).
- При некорректном вводе — сообщение об ошибке.
