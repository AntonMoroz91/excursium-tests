# Баг-репорты Excursium

## Баг №1: Фильтр «Место проведения» не сохраняется после обновления страницы

| Поле | Значение |
|------|----------|
| **Страница** | `https://excursium.com/ekskursii-dlya-shkolnikov/list` |
| **Серьёзность** | Незначительная (Minor) |
| **Приоритет** | Средняя (Medium) |
| **Окружение** | Windows 10, Google Chrome 147 |

**Шаги для воспроизведения:**
1. Открыть страницу со списком экскурсий.
2. Выбрать в фильтре «Место проведения» значение «Москва».
3. Обновить страницу (клавиша F5).

**Фактический результат:**  
Фильтр сбрасывается, отображаются все экскурсии.

**Ожидаемый результат:**  
Выбранный фильтр сохраняется (например, в URL или localStorage), список экскурсий остаётся отфильтрованным.

## Баг №2: Поле «Телефон» принимает более 12 символов

| Поле | Значение |
|------|----------|
| **Страница** | `https://excursium.com/register` |
| **Серьёзность** | Основная (Major) |
| **Приоритет** | Высокая (High) |
| **Окружение** | Windows 10, Firefox 136 |

**Шаги для воспроизведения:**
1. Открыть страницу регистрации.
2. В поле «Телефон» ввести 13 цифр (например, `1234567890123`).
3. Отправить форму.

**Фактический результат:**  
Регистрация проходит успешно.

**Ожидаемый результат:**  
Появляется сообщение об ошибке: «Не более 12 символов».
