# Баг-репорты Excursium

## Баг №1: Фильтр «Место проведения» не сохраняется после обновления страницы
- **Страница:** https://excursium.com/ekskursii-dlya-shkolnikov/list
- **Серьёзность:** Незначительная
- **Приоритет:** Средняя
- **Окружение:** Windows 10, Firefox 136
- **Шаги:**
  1. Открыть страницу экскурсий
  2. Выбрать фильтр «Место проведения» → «Москва»
  3. Обновить страницу (F5)
- **Фактический результат:** Фильтр сброшен, отображаются все экскурсии
- **Ожидаемый результат:** Фильтр сохраняется (в URL или localStorage)

## Баг №2: Поле «Телефон» принимает более 12 символов
- **Страница:** https://excursium.com/register
- **Серьёзность:** Основная
- **Приоритет:** Высокая
- **Окружение:** Windows 10, Firefox 136
- **Шаги:**
  1. Открыть страницу регистрации
  2. Ввести телефон из 13 цифр (например, 1234567890123)
  3. Отправить форму
- **Фактический результат:** Регистрация проходит успешно
- **Ожидаемый результат:** Ошибка «Не более 12 символов»
